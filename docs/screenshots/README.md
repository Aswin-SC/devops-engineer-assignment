# Screenshot Checklist

Capture and place screenshots here before final submission:

- `jenkins-pipeline.png`: successful Jenkins run with all stages.
- `kubernetes-workloads.png`: `kubectl get pods,deploy,svc,ingress -A`.
- `grafana-dashboard.png`: DevOps Assignment Platform dashboard.
- `alert-firing.png`: Prometheus or Grafana alert in firing state.
- `trivy-scan-results.png`: successful Trivy scan output.
